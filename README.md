# Smart Task Manager

A modern task manager built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Add, edit, and delete tasks.
- Priority, category, status, and due date support.
- Local storage persistence in the browser.
- Smart task suggestions for urgent and high-priority items.
- Filter by status, priority, category, search, and due-soon tasks.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
